# Accessibility Keyboard

Hands-free virtual keyboard using computer vision and NLP.

## Structure
- `/cv` – OpenCV + MediaPipe code
- `/nlp` – Text prediction models
- `/ui` – Streamlit or PyQt interface
- `/data` – Datasets
- `/docs` – Screenshots, notes
